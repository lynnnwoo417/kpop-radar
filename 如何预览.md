# 如何预览网站

由于浏览器安全限制，直接双击打开 `preview.html` 可能无法加载数据文件。

## 方法 1：使用 Python 启动本地服务器（推荐）

```bash
cd /Users/qixuan/kpop-schedule-web
python3 -m http.server 8000
```

然后在浏览器中打开：http://localhost:8000/preview.html

## 方法 2：使用 Node.js 的 http-server

```bash
# 安装 http-server（如果还没有）
npm install -g http-server

# 启动服务器
cd /Users/qixuan/kpop-schedule-web
http-server -p 8000
```

然后在浏览器中打开：http://localhost:8000/preview.html

## 方法 3：使用 VS Code Live Server

如果你使用 VS Code：
1. 安装 "Live Server" 扩展
2. 右键点击 `preview.html`
3. 选择 "Open with Live Server"

## 数据说明

预览文件会自动加载以下数据文件：
- `comebacks.json` - 回归数据
- `concerts.json` - 演唱会数据
- `fansigns.json` - 签售数据
- `festas.json` - 活动数据
- `melon_concerts.json` - Melon 演唱会数据
- `ticket_concerts.json` - 票务演唱会数据

如果数据文件加载失败，会自动使用示例数据。
